# Final-project-my-Python
🌍 Global Happiness Data Analysis
# 📌 Project Overview

This project analyzes the Global Happiness Dataset to explore factors that influence happiness levels across countries. Using Python data analysis and visualization libraries, the project examines relationships between happiness scores and indicators such as GDP, health, family support, and regional distribution.

The analysis includes data cleaning, exploratory data analysis (EDA), statistical summaries, and visualizations to identify patterns and trends in global happiness rankings.

🎯 Objectives
Clean and preprocess the dataset.
Handle missing values and duplicate records.
Generate descriptive statistics.
Identify the happiest and least happy countries.
Analyze the distribution of happiness scores.
Explore relationships between happiness and health indicators.
Visualize correlations among numerical features.
